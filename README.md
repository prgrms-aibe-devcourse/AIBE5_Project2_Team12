# IT-da

IT-da는 IT 업종 한정 프로젝트 의뢰인과 IT 프리랜서를 연결하는 AI 기반 매칭 플랫폼이다.
자유 텍스트 기반 프로젝트 입력을 AI 브리프로 구조화하고, 설명 가능한 추천으로 후보 비교와 선택을 돕는 것을 목표로 한다.
